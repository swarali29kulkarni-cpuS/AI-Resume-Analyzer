# AI Resume Analyzer 🚀

A Streamlit-based AI Resume Analyzer that extracts and analyzes resume content from PDF files.

## Features
- Upload PDF resume
- Extract text automatically
- Detect skills (Python, AWS, SQL, Machine Learning)
- Identify education background
- Detect project section
- Simple ATS-style scoring system

## Tech Stack
- Python
- Streamlit
- PyPDF2

## How to Run
```bash
py -m streamlit run app.py
